🌸 Supabase Configuration

Provide your Supabase credentials to persist memories and comments to your shared cloud database. Otherwise, everything is stored locally on your device!

💡 Setup SQL code for Supabase:

Run the following script in the SQL Editor of your Supabase dashboard:

-- Create a public table
create table posts (
  id uuid default gen_random_uuid() primary key,
  created_at timestamp with time zone default timezone('utc'::text, now()) not null,
  text_content text,
  image_url text
);

-- Enable RLS and allow public access
alter table posts enable row level security;
create policy "Allow public read" on posts for select using (true);
create policy "Allow public insert" on posts for insert with check (true);

-- IMPORTANT: Create a Storage bucket named 'memories' and set it to Public!
-- Add bucket policies allowing public READ and INSERT.
